## Building and Installing the Program

Do the following

```bash
mkdir -p build
cd build/
cmake ..
cmake --build .
cmake --install .
```

## Using the Program

Run the program generated in the repository's `bin` directory. From the `build/` directory, run

```bash
../bin/budget
```
